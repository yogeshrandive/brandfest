# SocietyBee Poster Generator

An internal tool for SocietyBee to generate branded social creatives.

## Setup

```bash
npm install
# Poppins fonts are downloaded automatically via postinstall
```

Copy `.env.example` to `.env.local` and fill in your keys.

## Deploy

Deploy to Vercel. Set env vars in the dashboard. Replace `public/brand/societybee-logo.png` with the real logo.
